What is JavaScript?

JavaScript is a programming language used for web development. It allows you to make pages interactive, like forms, animations, and dynamic updates.

What is the DOM?

The DOM (Document Object Model) is a tree-like structure representing HTML elements in memory, which JavaScript can manipulate.

What are events in JS?

Events are user interactions or browser triggers — like clicks, keypresses, or loads — that JavaScript can detect and respond to.
